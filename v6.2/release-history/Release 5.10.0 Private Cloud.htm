Known Issues

Following is the list of known issues with this release:

  1. [Logging] - In HA environment, if customer/tenant is deleted from the UI or using an API, exceptions like Stale file handle java.io.IOException: Stale file handle are mentioned in app server logs. This should never impact any functional behavior.
  2. [Logging] - With Move Storage operation, if the log directory is copied to a specific destination, the logJson directory is not copied along. If old json logs are required, the source logJson directory should be copied manually to destination before executing the Move Storage operation.
