Forgot password

Platform Users

If a user forgets their password, they can click Forgot your password on the login page. On the Forgot your password page, the user should enter the User Name and Email Address for a valid Platform user account:

If the information matches with any of the Platform records, Platform resets the password and sends a password activation link to user's email address. The user will be asked to change the password at the first login. For more information, see Change My Password.

Portal Users

The login component contains a Forgot Password link. This link redirects the user to Forgot your password page. On the Forgot your password page, the portal user should enter the Login Name and Email Address for the active portal user account.

An email confirming the password reset details is sent to the portal user's email address. The portal user will be asked to change the password at the first login. For more information, see Change My Password.
